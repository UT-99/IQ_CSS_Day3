Ques. What is box sizing?

In CSS box-sizing property is used to specify how to calculate the total height and width of an element.It controls the size of an element with a specified height and width.It allows you to include the padding and border within the total height and width of the element.

Ques. What is animation delay?

In CSS Animation delay is a property in CSS Animations this is used to specify an delay in the animation , the time specifed in delay is for seconds , after that time the animation will be started only when the web page is loaded.

Ques. Which property you will use to merge cells vertically in grid?

In CSS the cells of the grid can be merged vertically by using the child property grid-row-start and grid-row-end or by using the single parent property grid-row , this alows the grid-row to be started from one grid-row-line to another grid-row-line .

Ques. Which property you will use to merge cells horizontally in grid?

In CSS the cells of the grid can be merged horizontally by using the child property grid-column-start and grid-column-end or by using the single parent property grid-column , this alows the grid-column to be started from one grid-column-line to another grid-column-line .

Ques. Explain CSS box model

In CSS the components that can be depicted on the web page consist of one or more than one rectangular box.A CSS box model is a compartment that includes numerous assets, such as edge, border, padding and material. It is used to develop the design and structure of a web page. It can be used as a set of tools to personalize the layout of different components. According to the CSS box model, the web browser supplies each element as a square prism.

Ques. What is the difference between display none and visibility hidden?

In CSS display none and visibilty hidden is used to hide an element or make an element invisible on webpage but the difference between both the values is that display none will make the element invisible such that it was never existed in the web page as it does not acquire any space , but visibilty hidden will make the element hidden or invisible but it will use that space in the web page as an empty space will be shown on the web page .
